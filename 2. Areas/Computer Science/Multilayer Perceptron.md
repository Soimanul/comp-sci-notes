## Definition
It contains many different perceptrons arranged in different layers, out of which we only see the input and output; hence all the others are known as hidden layers

More formally it has:

1. **Input Layer**
2. **Hidden Layer(s)**
3. **Output Layer**

## Capability
Unlike a single [[Perceptron]], which can only classify linearly separable data (e.g., AND/OR gates), an MLP can solve non-linear problems (like the **XOR problem**) because of the hidden layers and non-linear [[Activation Functions]]s.
