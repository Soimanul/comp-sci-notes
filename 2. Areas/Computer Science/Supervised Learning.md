## Definition
A type of machine learning where the algorithm learns from a labeled dataset. The goal is to estimate a mapping function $f: X \to Y$ that can predict the output $y$ for new, unseen input data $x$.

## Core Components
- **Input ($x$)**: Feature vectors / predictors.
- **Output ($y$)**: Target labels.
- **Prediction ($\hat{y}$)**: Model's Prediction 
- **Training Set**: Pairs of $(x_i, y_i)$ used to estimate $f$.

## Data Types
- **Numerical**: Continuous or discrete numbers (e.g., height, temperature).
- **Categorical**: Discrete labels or classes (e.g., color, species).

## Common Tasks
1. **Classification**: Predicting discrete categories (e.g., [[Logistic Regression]]).
2. **Regression**: Predicting continuous values (e.g., [[Mean Squared Error (MSE)]]).
