## Definition
Language is treated as a structured symbolic system governed by explicit rules, where analysis starts from grammar and asks how sentences are generated and interpreted under those rules.

## What it focuses on
- **Generation:** how a finite rule system can produce an unbounded set of sentences.
- **Structure:** hierarchical constituent organization (phrases, nesting, recursion).
- **Compositionality:** meaning is built from smaller units via formal combinations.

## Core method
- Define a grammar and use it to determine whether a sentence is well-formed.
- Recover a sentence’s structural description via parsing (derivations/trees).

## What it treats as primary
Correctness is defined by derivability from rules, not by frequency or typicality in observed usage.
