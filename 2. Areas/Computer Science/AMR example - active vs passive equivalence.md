## Example pair
- “The boy broke the window.”
- “The window was broken by the boy.”

## What the example demonstrates
- Under CFG-style syntax, these sentences have different parse structures because active and passive constructions differ structurally.
- Semantically, they describe the same event with the same roles: the boy is the agent and the window is the affected object.

## AMR takeaway
AMR represents the shared meaning as a single underlying structure, making the equivalence explicit even when surface syntax differs.
