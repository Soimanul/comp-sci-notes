# Definition

ELIZA is an early natural language processing program created by **Joseph Weizenbaum** in **1966** at MIT. It is one of the first examples of a "chatterbot."

# Mechanism

ELIZA did not "understand" text. It used **Pattern Matching** and **Substitution** (script-based processing).

- **The DOCTOR Script:** The most famous implementation simulated a Rogerian psychotherapist.
    
- **Technique:** It scanned input for keywords (e.g., "mother"), decomposed the sentence, and reassembled it into a question or a generic response.
    

## Example Interaction

> **User:** "My mother hates me." **ELIZA:** "Who else in your family hates you?" (Pattern match on 'mother' -> family).

# Significance

- Demonstrated that a computer could hold a plausible conversation without any semantic understanding of the subject matter.
    
- Exposed the difference between **syntactic manipulation** (what ELIZA did) and **semantic understanding** (what users thought it did).
