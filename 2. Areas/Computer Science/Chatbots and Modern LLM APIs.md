> [!abstract]- TL;DR
> Modern chatbots have evolved from ELIZA's pattern matching through retrieval-based bots to LLM-based systems. ChatGPT (GPT-3.5/4 + RLHF) is the paradigm-shifting product. The OpenAI API offers three main interfaces: Chat Completions (stateless multi-turn), Responses API (stateful with built-in tools), and the Agents SDK (for multi-agent orchestration). Multiple major LLM providers now compete with compatible APIs.

## Overview

This session surveys the practical landscape for building chatbots in 2024+: what APIs exist, how they differ, and what capabilities modern LLM providers offer.
