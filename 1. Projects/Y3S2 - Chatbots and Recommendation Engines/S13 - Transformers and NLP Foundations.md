> [!abstract]- TL;DR
> Modern NLP is built on the Transformer architecture and pre-trained language models. The Transformer encodes sequences using self-attention (each token attends to all others), multi-head attention (running multiple attention heads in parallel), and positional encoding (injecting order information). Layer normalisation and residual connections stabilise training.

## Overview

Understanding the Transformer is essential for the chatbot half of this course. This session covers the architecture from first principles: from attention to multi-head attention to the full encoder/decoder stack.
